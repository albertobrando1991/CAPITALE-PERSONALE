# CAPITALE-PERSONALE

Progetto frontend/backend basato su Vite + TypeScript.

## Avvio locale

Installare dipendenze e avviare il client:

```bash
npm install
npm run dev
```

Server (se presente) si avvia con:

```bash
# nella cartella server (se serve)
npm run dev
```

## Pubblicazione su GitHub

1. Crea un repository su GitHub (o usa `gh repo create`).
2. Aggiungi il remote e fai push:

```bash
git remote add origin https://github.com/TUO_USERNAME/TUO_REPO.git
git branch -M main
git push -u origin main
```

Sostituisci `TUO_USERNAME/TUO_REPO` con il tuo repository reale.
